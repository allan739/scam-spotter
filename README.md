# Scam Spotter — Trust Your Gut

An interactive quiz app that helps you learn to spot common scams before they catch you.

## Features

- 🎯 **20 Real-life Scenarios** — Learn warning signs from common scam tactics
- ✅ **Instant Feedback** — Get clear explanations after each answer
- 📊 **Progress Tracking** — Visual progress dial and score
- 📱 **PWA Support** — Works offline and can be installed as an app
- 🎨 **Beautiful Design** — Clean, accessible interface
- 💾 **Score Memory** — Tracks your best score

## How It Works

1. Read a real-life scenario
2. Decide: Is it a SCAM or is it SAFE?
3. Get immediate feedback and explanation
4. Learn the warning signs to protect yourself
